Set Up
------

###### For 24 Bit Terminals
```VimL
set termguicolors
colorscheme monrovia
```

###### For GUI
```VimL
colorscheme monrovia
```

###### For 8/16/256 Terminals ( Use Matching Terminal Xresources or iTerm )
```VimL
set t_Co=16
colorscheme monrovia
```
